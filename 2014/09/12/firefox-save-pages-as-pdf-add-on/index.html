firefox save pages as pdf add-on

firefox里，有一个比较好用的保存网页为pdf的插件: Print pages to Pdf

特点

  1. 插件使用本地浏览器，将网页生成pdf文件，不同于其他插件，使用服务器资源；
  2. 无广告；
  3. 转换速度快，而且支持中文内容。

缺点

  1. 不支持64位的firefox, 64位的不能正确保存文件；
  2. 只支持windows系统；
  3. 文件名不能自动识别中文，只能保存之后再手动重命名为中文。
