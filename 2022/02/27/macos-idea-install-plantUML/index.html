Macbook（M1）Idea安装PlantUML插件

前言

最近发现PlantUML是一个非常好用的画UML图的工具；画类图、流程图、状态图、时序图非常方便，而且语法简单；

记录一下MacBook（M1）系统下给IntelliJ Idea安装PlantUML插件流程。

开始

安装brew

  1. 使用国内的镜像安装brew：

    /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
# 使用1的源
    • 参考：https://zhuanlan.zhihu.com/p/372576355

  2. 使用brew安装graphviz：

    brew install graphviz
Idea安装PlantuUML插件

  1. 打开Idea，Preferences>Plugins>Marketplace，搜索：PlantUML，安装4个插件：

  2. 安装完成后，点击Idea右侧的“PlantUML”，选择：Open Settings：

  3. 修改“Graphviz dot execute”，值为：/opt/homebrew/bin/dot：

  4. 点击“OK”，完成。
