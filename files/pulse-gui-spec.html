Pulse GUI 语言设计规范

一、核心理念

GUI 由三个核心概念组成：

GUI = 自身 × 世界 × 关系

自身：是什么/有什么（内容）
世界：在哪里/怎么动（时空）
关系：与谁相关（连接）

二、三元 × 三层

概念 原子 组合 聚合
自身 node component view
世界 space viewport scene
关系 relation group tree

三、自身（是什么/有什么）

3.1 node（原子）

最小可见单元，不可再分：

<Node
    shape="rect"
    size="100 50"
    fill="red"
    stroke="1 black"
    text="Hello"
    image="icon.png"
    font="16 bold"
    opacity="0.8"
/>

3.2 component（组合）

node 的组合，可复用：

component Button

<props>
    label: String
    disabled: bool = false
</props>

<state>
    mode: idle | pressed
</state>

<template>
    <Node
        shape="rect"
        size="80 40"
        fill="{{ mode == pressed ? theme.dark : theme.primary }}"
        text="{{ label }}"
    />
</template>

3.3 view（聚合）

component 的组合，页面级别：

view OrderPage

<props>
    userId: String
</props>

<state>
    orders: List<Order> = []
    loading: bool = true
</state>

<handle>
    ViewLoad { } {
        send Order.Fetch { userId }
    }
    
    Order.Fetched { orders } {
        set orders = orders
        set loading = false
    }
</handle>

<template>
    <Scene>
        <Node if="loading" text="加载中..." />
        <OrderCard for="order in orders" :order="order" />
    </Scene>
</template>

四、世界（在哪里/怎么动）

4.1 space（原子）

节点的位置、旋转、缩放：

<Node
    position="100 200"
    rotation="45deg"
    scale="1.5"
/>

4.2 viewport（组合）

可视窗口，决定用户看到什么：

<Viewport size="320 240" offset="0 100">
    <Scene>...</Scene>
</Viewport>

4.3 scene（聚合）

场景空间，可以比视窗大：

<Scene size="1000 1000">
    <Node position="500 500" />
</Scene>

4.4 变化（怎么动）

动画三要素（无时间）：

要素 说明 值
频率 快慢（相对） instant / fast / normal / slow
程度 变化幅度 属性变化
循环 重复次数 1 / n / infinite 
<transition>
    idle -> pressed {
        frequency: fast
        magnitude: {
            scale: 1 -> 0.95
            fill: theme.primary -> theme.dark
        }
        loop: 1
    }
    
    loading {
        frequency: normal
        magnitude: {
            rotation: 0deg -> 360deg
        }
        loop: infinite
    }
</transition>

五、关系（与谁相关）

5.1 relation（原子）

父子连接、层级：

<Node parent="container" layer="1" />

5.2 group（组合）

组织分组、布局：

<Group layout="row gap:8 align:center">
    <Button label="保存" />
    <Button label="取消" />
</Group>

5.3 tree（聚合）

层级结构：

<Tree>
    <Node id="root">
        <Node id="header" />
        <Node id="content" />
        <Node id="footer" />
    </Node>
</Tree>

六、属性汇总

6.1 自身属性

属性 作用 示例
shape 形状 rect / circle / path
size 尺寸 100 50 / fill
fill 填充 red / #007AFF / theme.primary
stroke 描边 1 black
text 文字 "Hello"
image 图片 "icon.png"
font 字体 16 bold
opacity 透明度 0.8

6.2 世界属性

属性 作用 示例
position 位置 100 200
rotation 旋转 45deg
scale 缩放 1.5
visible 可见 true / false

6.3 关系属性

属性 作用 示例
parent 父节点 "container"
layer 层级 1 / "overlay"
layout 布局 row / column
align 对齐 center / start / end
gap 间距 8

七、模板语法

语法 作用 示例
{{ value }} 文本插值 <Node text="{{ name }}" />
:prop="value" 属性绑定 :disabled="loading"
::prop="value" 双向绑定 ::value="input"
@event="handler" 事件绑定 @click="HandleClick"
if 条件渲染 <Node if="loading" />
else-if 条件渲染 <Node else-if="error" />
else 条件渲染 <Node else />
for 循环渲染 <Node for="item in items" />
:key 循环键 :key="item.id"

八、主题

theme Light {
    primary: #007AFF
    dark: #0056B3
    background: #F2F2F7
    text: #333333
    border: #E5E5EA
}

theme Dark extends Light {
    background: #1C1C1E
    text: #FFFFFF
}

九、关键字汇总

关键字 归属 作用
node 自身-原子 基础节点
component 自身-组合 组合组件
view 自身-聚合 页面视图
viewport 世界-组合 可视窗口
scene 世界-聚合 场景空间
group 关系-组合 分组
tree 关系-聚合 层级树

十、完整示例

10.1 Button 组件

component Button

<props>
    label: String
    disabled: bool = false
    onClick: Event
</props>

<state>
    mode: idle | pressed
</state>

<handle>
    Press { } {
        if !disabled { set mode = pressed }
    }
    
    Release { } {
        set mode = idle
        send onClick
    }
</handle>

<template>
    <Node
        shape="rect"
        size="80 40"
        fill="{{ disabled ? theme.border : theme.primary }}"
        text="{{ label }}"
        font="14 bold"
        @pressIn="Press"
        @pressOut="Release"
    />
</template>

<transition>
    idle -> pressed {
        frequency: fast
        magnitude: { scale: 1 -> 0.95 }
    }
    
    pressed -> idle {
        frequency: fast
        magnitude: { scale: 0.95 -> 1 }
    }
</transition>

10.2 OrderPage 视图

view OrderPage

<state>
    orders: List<Order> = []
    loading: bool = true
</state>

<handle>
    ViewLoad { } {
        send Order.Fetch { }
    }
    
    Order.Fetched { orders } {
        set orders = orders
        set loading = false
    }
    
    Delete { orderId } {
        send Order.Delete { orderId }
    }
    
    Order.Deleted { orderId } {
        set orders = orders.filter { it.id != orderId }
    }
</handle>

<template>
    <Viewport size="fill">
        <Scene layout="column gap:16 padding:16">
            <Node text="我的订单" font="24 bold" />
            
            <Node if="loading" text="加载中..." />
            
            <Group else layout="column gap:12">
                <OrderCard
                    for="order in orders"
                    :key="order.id"
                    :order="order"
                    @delete="Delete { orderId: order.id }"
                />
            </Group>
            
            <Button label="新建订单" @click="Create" />
        </Scene>
    </Viewport>
</template>

<transition>
    enter {
        frequency: normal
        magnitude: { opacity: 0 -> 1 }
    }
</transition>

十一、设计原则

原则 说明
三元 自身 × 世界 × 关系
三层 原子 → 组合 → 聚合
无时间 频率相对，渲染层决定实际速度
事件驱动 handle 响应，send 触发
声明式 描述”是什么”，不是”怎么做”

十二、总结

自身定义内容，世界承载位置，关系组织结构，事件驱动变化。
